Maken Joomla! testomgeving

Herman in 't Veen

Met Joomla! kun je relatief eenvoudig websites bouwen, maar voor je iets kunt bouwen heb je een ontwikkelomgeving nodig. Zo'n intwikkelomgeving bestaat uit een webserver waarop je Joomla! kunt installeren, dit artikel legt uit hoe je die opzet.

Als webserver gebruik ik een virtuele machine, die ik heb gemaakt via VirtualBox.

  • Download en installeer Oracle Virtual Box
  • Download het operating system (ISO), ik gebruik zelf Ubuntu Linux server 14.04.1 LTS
  • Maak een nieuwe Virtuele machine aan in VirtualBox, kies name (Joomla server), Type (Linux) en Version (32 of 64 bit van jouw PC/Mac)
  • Kies Base Memory naar wens (16+ MB)
  • Kies bij Storage / Controller IDE /Add CD/DVD device het door jou gedownloade operating system bestand
  • Kies bij Network Bridged adapter als je een thuisnetwerk hebt, werk je stand alone kies dan Host-onkly adapter
  • Start je VirtualBox Joomla server, de server booot het door jou gekozen operating system van de CD

Voor Ubuntu server heb ik de volgende keuzes gemaakt: 

  • Language: English
  • Install ubuntu server : Kies “English” enter
  • Kies F4 – (OS X : Toets fn + F4)
  • Kies: Install a minimal virtual machine – enter
  • Language: Engels
  • Country: Netherlands
  • Locale: US
  • Keyboard: USA
  • Hostname : joomlaserver
  • User: <kies usernaam> – login <kies login naam> en <kies wachtwoord>
  • Geen encryptie
  • Geen proxy
  • Partitioning : guided – use entire disk – accept defaults & write to disk
  • Geen o http proxy
  • Install security updates automatically
  • Package software install: LAMP only (Linux Apache MySql en PHP)
  • MySql root password : <kies wachtwoord>
  • Grub boot loader install OK
  • Finish install & reboot

Nu herstart de joomlaserver en je krijgt een login prompy, log in met je gekozen login naam en wachtwoord.
Type het commando: ifconfig
Noteer het IP-adres waarop je server draait: dit staat achter "inet addr:" en begint typisch met 192.168.xxx.yyy
Je hebt dit IP-adres nodig als je Joomla! wil installeren op je joomlaserver.

De server is nu gereed om hier verdere software (Joomla!) op te installeren.
